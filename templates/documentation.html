Qrious API Documentation

Dynamic QR generation, analytics, webhooks, WiFi codes, contact QR, and barcode services. Build powerful QR-driven experiences with modern APIs.

Base URL

All requests use HTTPS. This is the production endpoint. 

https://qr.soltryxsolutions.com

Authentication

Public endpoints available for testing. Use API keys for production and rate control. 

POST /api/key/create
Form:
owner (string)
limit (int, default 5000)

POST /api/generate-qr (dynamic)

Generates a dynamic QR code that redirects through the service. Destinations can be edited, scanned, and analyzed later.

Features

  • Scan analytics
  • Geolocation stats
  • Expiration & password protection
  • Editable destination
  • Webhook notifications

Use Cases

  • Marketing campaigns
  • Product tracking
  • Digital business cards
  • Secure link sharing
Parameter Type Required Description
datastring (URL)YesDestination URL
expires_inintNoExpiration (minutes)
passwordstringNoAccess key
logofile/base64NoBrand overlay
webhookstringNoScan event URL
templatestringNoTag/identifier

GET /r/{qr_id}

Dynamic redirect endpoint. Tracks scan events and applies protection when configured.

Returns HTTP 410 if expired, HTTP 403 if password is required.

GET /api/qr/{qr_id}/stats

Analytics and history for each QR code.

Field Description
scan_countTotal scans
unique_scansUnique IP scans
countriesGeo distribution
versionsEdit history
tagsCustom tags

Webhooks

If a webhook URL is provided, scan events will POST analytics data. 

POST webhook_url
{
  "qr_id": "...",
  "ip": "hashed_ip",
  "geo": {...},
  "device": "mobile|desktop|mac|unknown",
  "time": "timestamp"
}

POST /api/generate-wifi-qr

Auto-connect WiFi QR codes.

Parameter Type Required Description
ssidstringYesNetwork name
passwordstringYesNetwork password
securitystringNoWPA or nopass

POST /api/generate-barcode

Generate Code128 barcodes for product codes or text.

Contact QR (vCard)

Instant contact import using vCard format. 

BEGIN:VCARD
VERSION:3.0
FN:Full Name
TEL:+63XXXXXXXXXX
EMAIL:email@example.com
END:VCARD
