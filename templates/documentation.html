QR & Barcode API Documentation

Generate QR codes with optional branding and WiFi auto-connect support.

Base URL

https://qr.soltryxsolutions.com

Authentication

Public access available. API key support on request.

POST /generate-qr

Generate a QR code from data with optional logo branding.

Request

Content-Type: application/x-www-form-urlencoded

Parameter Type Required Description
data string Yes QR content (URL, WiFi, vCard, text)
logo string (base64) No Base64 image placed in QR center. If omitted → default static/logo.png.

Example (default logo)

curl -X POST http://127.0.0.1:8000/api/generate-qr \
-d "data=https://site.com" --output qr.png

Example (custom logo)

curl -X POST http://127.0.0.1:8000/api/generate-qr \
-d "data=https://site.com&logo=$(base64 -w 0 logo.png)" --output qr.png

High error correction ensures scanability even with logo overlay.

POST /api/generate-wifi-qr

Generate WiFi QR codes that auto-connect on scan.

Parameter Type Required Description
ssid string Yes WiFi network name
password string Yes WiFi password
security string No WPA or nopass (default: WPA) 
curl -X POST http://127.0.0.1:8000/api/generate-wifi-qr \
-H "Content-Type: application/x-www-form-urlencoded" \
-d "ssid=test&password=test123&security=WPA" --output wifi.png

QR content: WIFI:S:test;T:WPA;P:test123;;

POST /generate-barcode

Generate Code128 barcodes from product codes or text.

Parameter Type Required Description
data string Yes Content to encode 
curl -X POST https://qr.soltryxsolutions.com/generate-barcode \
-d "data=123456789012"
